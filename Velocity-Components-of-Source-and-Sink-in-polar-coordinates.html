Velocity Components of Source and Sink in Polar Coordinates

Stream function comes from the continuity equation \(\color{#2b8ae0}\nabla\cdot V=0 \)

Potential function comes from irrotationality equation \(\color{#bb3e19}\nabla \times V = 0\)

I'm going to use colours to keep the distinction clear.

Stream function

To find velocity components through stream function we are going to use the EXACT ODE method. $$\color{#2b8ae0}\nabla\cdot V=0 $$ $$\color{#2b8ae0} \frac{\partial u_r}{\partial r} + \frac{1}{r}\frac{\partial u_{\theta}}{\partial \theta} = 0 $$ Let there be some function \(\psi\), derivative of which w.r.t. \(\theta \) is : $$\color{#2b8ae0} u_r = \frac{1}{r}\psi_{_\theta} $$ At the same time the derivative of \(\psi\) w.r.t. \(r\) is: $$\color{#2b8ae0} u_{\theta} = -\psi_{_r} $$
[Click to show details] How to come to these assumptions by trial and error
You don't have to memorize anything here. Let's get to the above functions by trial and error.

Let's begin solving this PDE using the exact method. Let there be some function psi (\(\psi\)) where if we take it's derivative w.r.t. theta we get \(u_r\): $$ \frac{\partial \psi}{\partial\theta} = u_r $$ Also, a derivative of \(\psi\) w.r.t. \(r \) would be \(u_{\theta}\): $$ \frac{\partial \psi}{\partial r} = u_{\theta} $$ NOTE: This is not correct, but that's ok. This is how we get to the correct version. Bear with me.

For this to be an exact PDE the following must be true: $$ \frac{\partial }{\partial r}\frac{\partial \psi}{\partial\theta} =\frac{\partial }{\partial \theta}\frac{\partial \psi}{\partial r} $$ Let us see if this is the case by plugging our assumptions into the PDE.

For this to be an exact ODE (or PDE in this our case), if we plug the above assumptions into mass continuity equation the following must be true, $$ \frac{\partial}{\partial r}\psi_{_\theta} \equiv \frac{\partial}{\partial \theta}\psi_{_r} $$ Let's plug it in and see if it's true, and if it's not, what we can do about it. $$\text{PDE: } \frac{\partial u_r}{\partial r} + \frac{1}{r}\frac{\partial u_{\theta}}{\partial \theta} = 0 $$ $$ \frac{\partial}{\partial r}\frac{\partial \psi}{\partial\theta} + \frac{1}{r}\frac{\partial}{\partial \theta}\frac{\partial \psi}{\partial r} = 0 $$ $$ \frac{\partial}{\partial r}\frac{\partial \psi}{\partial\theta} =- \frac{1}{r}\frac{\partial}{\partial \theta}\frac{\partial \psi}{\partial r} $$ in other words, $$ \frac{\partial}{\partial r} \psi{_\theta} \ne- \frac{1}{r} \frac{\partial}{\partial \theta} \psi{_r} $$ But that's ok. We can see what needs to be done. To make this equality true, let's make the following adjustments: $$ u_r = \frac{1}{r} \psi{_\theta}, \ \ \ \ u_{\theta}=-\psi_{_r} $$ Let us plug these new assumptions into the PDE. $$ \frac{\partial u_r}{\partial r} + \frac{1}{r}\frac{\partial u_{\theta}}{\partial \theta} = 0 $$ $$ \frac{\partial}{\partial r}\frac{1}{r}\frac{\partial \psi}{\partial\theta} + \frac{1}{r}\frac{\partial}{\partial \theta}\biggl(-\frac{\partial \psi}{\partial r}\biggr) = 0 $$ $$ \frac{\partial}{\partial r}\frac{1}{r}\psi{_\theta} - \frac{1}{r}\frac{\partial}{\partial \theta} \psi{_r} = 0 $$ $$ \frac{\partial}{\partial r}\frac{1}{r}\psi{_\theta} = \frac{1}{r}\frac{\partial}{\partial \theta} \psi{_r} $$ $$ \frac{\partial}{\partial r}\psi_{_\theta} \equiv \frac{\partial}{\partial \theta}\psi_{_r} $$ Therefore, these assumptions are correct, and we can solve our PDE using the exact method. Except, we were not solving it. We were just looking for the velocity components.

Potential function

To find velocity components through potential function is a bit easier. $$\color{#bb3e19}\nabla \times V = 0$$ $$\color{#bb3e19}V=\nabla \phi $$ $$\color{#bb3e19} u_r \hat r +u_{\theta} \hat \theta = \hat r \frac{\partial \phi}{\partial r} + \frac{1}{r}\hat \theta \frac{\partial \phi}{\partial \theta} $$ Left Hand Side (LHS) is equal to the Right Hand Side (RHS). Both sides are vectors. Therefore, we can conclude that, $$\color{#bb3e19} u_r = \frac{\partial \phi }{ \partial r} \ \ \ \ \ u_{\theta}=\frac{1}{r}\frac{\partial \phi}{\partial \theta} $$ FYI, we've just paired the \(\hat r\) and \(\hat\theta\) components to their counterparts.

Velocity components in polar coordinates

Bring it all together, we can now define our velocity components of the fluid in a steady, incompressible, irrotational flow in the polar coordinate system. $$ u_r \color{#2b8ae0} = \frac{1}{r}\psi_{_\theta} \color{#bb3e19} = \frac{\partial \phi }{ \partial r} $$ $$ u_{\theta} \color{#2b8ae0} = -\psi_{_r} \color{#bb3e19} = \frac{1}{r}\frac{\partial \phi}{\partial \theta} $$
